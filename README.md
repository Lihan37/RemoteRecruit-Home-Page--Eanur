# RemoteRecruit Home Page

React + Vite + Tailwind CSS implementation of the RemoteRecruit landing page.

## Project Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

## Frameworks And Libraries

- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React
- ESLint

## Known Issues And Limitations

- Some visual sections are matched from exported image assets, so exact responsiveness may need final QA against the Figma file.
- The footer social links currently use placeholder `#social` URLs.
- The FAQ content is mock placeholder copy.
- npm reports dependency audit warnings; no forced audit fix has been applied to avoid breaking changes.
